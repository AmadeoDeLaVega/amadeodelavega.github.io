This manuscript explores whether neural networks can spot malicious network traffic more effectively than rule-based filtering alone. Instead of relying only on hand-crafted firewall signals, the project studies how to represent packet payloads as model inputs and train deep learning systems to classify traffic as benign or malicious.

For undergraduate CS students, this paper is a useful bridge between cybersecurity and machine learning. It shows how raw network data becomes features, how model design choices affect real-world deployment, and why reproducibility and dataset quality matter just as much as model accuracy.
